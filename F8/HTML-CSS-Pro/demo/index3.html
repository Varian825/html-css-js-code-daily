Ví dụ: bạn có thể đánh dấu một đoạn văn bản với HTML như này:

<p id="paragraph">What a beautiful day today!</p>

Sau đó, bạn có thể đặt kích thước và màu sắc cho đoạn văn bản với CSS

#paragraph {
    font-size: 18px;
    color: red;
}

Cuối cùng, nếu bạn muốn nhấn vào đoạn văn bản thì nội dung của nó sẽ thay đổi, thêm code JavaScript sau. 

const p = document.getElementById('paragraph');
p.onclick = function() {
    this.innerText = 'You have CLICKED on the text!';
};
